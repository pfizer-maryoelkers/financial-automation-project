# financial-automation-project
This repository contains an end-to-end ETL pipeline for automating financial data processing. The solution integrates SharePoint for file retrieval and upload, and uses Python for data extraction, transformation, and loading into a standardized financial template.


## Project Setup:

In command prompt, run the following commands to get set up with the repository

1. Clone the repository
```
git clone <repo-url>
cd <repo-folder>
```

2. Create a virtual python environment
```
python -m venv venv 
```

After creating the environment, activate the environment by running:
```
venv\Scripts\activate
```
*Note:* Should see (venv) in your terminal - VSCode does this automatically

3. Install project dependencies
```
pip install -r requirements.txt
```
